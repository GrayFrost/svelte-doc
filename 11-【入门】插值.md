```
<slot><!-- 可选回调 --></slot>
<slot name="x"><!-- 可选回调 --></slot>
<slot prop={value}></slot>
```


Slot可以零次或多次渲染，并且可以通过prop属性将值传回父级,父级使用 `let:`指令将值暴露到slot模板。

通常适用的速记规则：`let:item` 等效于 `let:item={item}`，并且 `<slot {item}>` 等效于 `<slot item={item}>`


```
<!-- App.svelte -->
<FancyList {items} let:item={item}>
	<div>{item.text}</div>
</FancyList>

<!-- FancyList.svelte -->
<ul>
	{#each items as item}
		<li class="fancy">
			<slot item={item}></slot>
		</li>
	{/each}
</ul>
```
